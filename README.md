## Getting Started

### FrontEnd
1. Install packages

```
yarn
```

2. Run project

```
yarn start
```

### BackEnd
1. Install packages

```
flask
```
2. Place cmd in server folder

```
cd server
```

3. Activate virtual env

```
.\env\scripts\activate.bat
```

4. Run project

```
flask run
```
